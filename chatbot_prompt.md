### 챗봇 제작 요청 프롬프트 (학생용)

**제목: Node.js와 `gemini` CLI를 이용한 간단한 웹 챗봇 제작 요청**

안녕하세요. 지금부터 간단한 웹 기반 챗봇 애플리케이션 제작을 요청합니다. 아래 요구사항을 정확히 따라서 구현해주세요.

**1. 최종 목표**
사용자가 웹 화면에서 메시지를 입력하면, 백엔드 서버가 `gemini` CLI 명령어를 실행하여 그 결과를 다시 웹 화면에 표시해주는 간단한 챗봇을 만들어주세요.

**2. 기술 요구사항**
- **백엔드**: **Node.js**와 **Express** 프레임워크를 반드시 사용해야 합니다. Python, Flask 등 다른 기술은 사용하지 마세요.
- **프론트엔드**: 순수한 **HTML, CSS, JavaScript**로 구성해주세요. React, Vue 등의 프론트엔드 라이브러리는 필요 없습니다.

**3. 아키텍처**
이 애플리케이션은 두 부분으로 구성됩니다.
- **프론트엔드 (클라이언트)**: 사용자가 보는 웹 화면입니다. 브라우저의 보안 정책상, 프론트엔드 JavaScript는 로컬 컴퓨터의 셸 명령어를 직접 실행할 수 없습니다.
- **백엔드 (서버)**: 프론트엔드의 요청을 받아 셸 명령어를 대신 실행하고 결과를 돌려주는 중간 다리 역할을 합니다.

**4. 백엔드 서버 상세 요구사항**
- Node.js와 Express를 사용하여 웹 서버를 구축해주세요.
- `/api/chat`이라는 주소로 `POST` 요청을 받는 API 엔드포인트(endpoint)를 만들어주세요.
- 이 엔드포인트는 프론트엔드에서 보낸 사용자 메시지(JSON 형식)를 받습니다.
- 메시지를 받으면, 아래와 같은 형식의 **`gemini` 셸 명령어를 실행**해야 합니다.
    - **명령어 형식**: `gemini -y -p "<사용자 메시지>"`
    - **`gemini -y -p` 상세 설명**:
        - `gemini`: 실행할 프로그램의 이름입니다.
        - `-y`: "YOLO mode"를 의미하며, 명령어 실행 중 발생하는 모든 확인 요청을 자동으로 승인하는 중요한 옵션입니다. 반드시 포함해주세요.
        - `-p "..."`: 사용자 메시지를 프롬프트로 전달하는 옵션입니다. 메시지는 반드시 **큰따옴표(`"`)**로 감싸야 합니다. 만약 사용자 메시지 자체에 큰따옴표가 포함되어 있을 경우, 명령어 오류가 발생하지 않도록 **적절히 이스케이프(escape) 처리**하는 로직을 포함해야 합니다.
- 명령어 실행이 완료되면, 그 결과(stdout)를 프론트엔드에 JSON 형식으로 응답해주세요.

**5. 프론트엔드 화면 상세 요구사항**
- 대화 내용이 표시될 채팅창, 메시지 입력 필드, 전송 버튼으로 구성된 간단한 UI를 만들어주세요.
- 사용자가 메시지를 보내면, JavaScript의 `fetch` 함수를 사용하여 백엔드 서버의 `/api/chat` 엔드포인트로 `POST` 요청을 보내세요.
- 이때, API 서버의 주소를 `http://localhost:3000` 과 같이 고정하지 말고, **상대 경로(`/api/chat`)**를 사용하여 서버 포트 번호가 변경되어도 문제가 없도록 만들어주세요.
- 서버로부터 받은 응답을 채팅창에 표시해주세요.

**6. 최종 결과물 및 실행 안내**
- 모든 코드 구현이 완료된 후, 사용자가 직접 서버를 실행할 수 있도록 아래와 같이 명확하고 친절한 안내 문구를 **반드시** 마지막에 포함해주세요.

> **[서버 실행 방법 안내]**
>
> 1.  **PowerShell(파워셸) 열기**: 시작 메뉴에서 `PowerShell`을 검색하여 실행합니다.
> 2.  **프로젝트 폴더로 이동**: 아래 명령어를 복사하여 PowerShell에 붙여넣고 엔터를 누르세요.
>     `(서버 파일이 있는 정확한 전체 경로를 여기에 명시해주세요)`
> 3.  **서버 실행**: 아래 명령어를 입력하고 엔터를 누르면 서버가 시작됩니다.
>     `node server.js`
>
> 이제 웹 브라우저를 열고 `http://localhost:(서버 포트 번호)` 주소로 접속하면 챗봇을 사용할 수 있습니다. 서버를 종료하려면 PowerShell 창에서 `Ctrl` + `C`를 누르세요.
