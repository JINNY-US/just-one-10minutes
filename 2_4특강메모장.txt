1. 윈도우키 -> powershell 검색 및 실행
2. winget install Git.Git 
명령어 복사 붙여넣기로 입력 후 엔터 
-> 모든 원본 사용 약관에 동의하십니까? y입력후 엔터
설치 성공
3. winget install OpenJS.NodeJS
명령어 복사 붙여넣기로 입력 후 엔터
설치성공
4. powershell창을 닫고 다시 윈도우키 -> powershell 입력 재실행
이후 아래 명령어를 실행해서 설치 확인
git --version
node --version
npm --version
(정상 설치시 3가지 모두 버전이 표시됨)
5. npm --version 입력시 "npm : 이 시스템에서 스크립트를 실행할 수 없으므로" 등에 에러 발생시
-> 윈도우키 -> powershell검색 -> 오른쪽의 관리자권한으로 다시 실행
아래의 명령어를 입력
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
실행 규칙 변경 나오면 a입력 후 엔터
이후 관리자로 실행한 powershell 창을 닫고 
powershell '일반권한으로 재실행
6. 버전을 확인한 powershell 창에 아래의 명령어 입력후 실행
npm install -g @google/gemini-cli
설치 완료시 gemini --version 입력으로 설치 확인
7. gemini -y를 입력 후 엔터 -> Get started 화면
(만약에 화면이 멈춘것 같다 -> 화면 클릭 후 스페이스바 한번 클릭)
login with goolge에 초록색 점이 찍힌 상태이다 -> 엔터키를 누르기
(엑세스 허용을 물을경우 엑세스 허용)
내 구글 계정으로 로그인 진행  -> 인증 완료창이 떴으면 크롬 닫기
You have successfully logged in with Google. 표시되었다면 r키 누르기
-> Type your message 공간이 나오면 성공

8. Type your message 공간에 내가 원하는 요청사항을 입력하고 엔터를 눌러서 작동

9. 사용할 새로운 폴더 만들기 -> 우클릭 새로만들기 -> 텍스트 문서 만들기
-> 텍스트 문서의 이름(새 텍스트 문서.txt)을 GEMINI.md로 변경(확장자포함, .txt로 끝나지 않도록) -> 우클릭을 눌러서 연결프로그램 메모장으로 열기
-> 메모장에 내가 원하는 규칙(ai의 이름 설정, 한국어로 대답해줘 등)
*주의 : 폴더에서 보기 -> 파일확장명 체크박스 활성화 -> GEMINI.md.txt 등으로 되어 있다면 .txt를 제거

10. gemini실행중인 창에서 /quit입력 (아래쪽에 exit the cli가 떠야 정상) 후에 엔터 ->  Goodbye!가 나오며 gemini종료

11. 현재 powershell 창에 cd(띄고)"내가 만든 폴더 경로" 입력후 엔터
-> 현재 powershell 터미널에 내가 입력한 폴더 경로로 현재 위치 변경 확인
-> 그상태에서 다시 gemini -y 입력으로 실행
-> Type your message 공간 왼쪽 위에  1 GEMINI.md file 이라고 떠있으면 현재 인식 성공

11. 테스트(Type your message 공간에 입력):
'현재 폴더에 간단한 새해 인삿말을 적어서 txt파일로 저장해줘' 
12. 테스트2
'간단한 카페 렌딩페이지를 섹션을 4개정도 구성해서 html파일로 만들어줘'
-> 이후 완성된 html을 수정요청해보기

13. 챗봇 제작 실습
 /clear 입력 -> 채팅방 나가기(현재 gemini와 대화중이던 채팅방 나가기 = 초기화)
-> chatbot_prompt.md 다운받아서 폴더에 넣어두기
-> chatbot_prompt.md 파일을 폴더에서 powershell 터미널 창으로 드래그
-> 현재 powershell 터미널 창에 파일경로 바로 붙여넣기 가능해짐
-> 그다음에 "이거를 읽고 프롬프트대로 챗봇을 별도 폴더에 만들어줘"
-> keep trying 관련 질문이 나오면 엔터 눌러서 keep trying 선택
-> allow 관련 질문이 나오면 아랫방향키 눌러서 2번 선택후 엔터로 선택

14. [서버 실행 방법 안내] 가 나오면 가이드대로 진행 후 주소 접속
- powershell 새창으로 실행(기존 gemini 실행중이던 powershell창은 끄지않고 추가로 새 창을 열기)
2. 거기에 cd C:\어쩌구.... 등의 가이드에 나온거 입력
3.  node server.js 입력 후 실행
4. 가이드에 적힌 주소 접속

챗봇 성능 테스트 진행 -> 이후 원하는 대로 수정하며 완성해보기
(ex:챗봇에서 요청사항을 보냈을때 응답 대기하는 애니메이션 로딩 등이 추가됐으면 좋겠어, 컬러 테마를 보라색으로  했으면 좋겠어 등등)


/clear가 채팅방 나가서 채팅 초기화
/resume 으로 채팅방 목록에서 다시 들어가고 싶은 채팅방에 들어가서 이어서 대화 가능
