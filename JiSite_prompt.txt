제목: Node.js + Express + gemini CLI 기반 Notion 스타일 고급 글쓰기 & 아카이빙 웹앱 제작 요청

안녕하세요.
Notion과 유사한 고급 글쓰기 및 아카이빙 웹 애플리케이션 제작을 요청합니다.
아래 요구사항을 정확히 따라 전체 프로젝트 코드를 구현해주세요.

------------------------------------------------------------
1. 최종 목표
------------------------------------------------------------

다음 기능을 포함한 웹 애플리케이션을 만들어주세요:

1) 마크다운 기반 문서 작성
2) 드래그 앤 드롭 블록 편집
3) 문서 저장 및 수정
4) 문서 검색 기능
5) gemini CLI를 활용한 AI 기능 (요약 / 정리 / 태그 생성)

데이터는 초기 버전에서는 JSON 파일 기반으로 저장합니다.

------------------------------------------------------------
2. 기술 요구사항 (반드시 준수)
------------------------------------------------------------

[백엔드]
- Node.js + Express만 사용
- Python 사용 금지

[프론트엔드]
- HTML + CSS + Vanilla JavaScript만 사용
- React, Vue, Next.js 사용 금지
- 외부 CDN 라이브러리는 허용

5. 백엔드 API 요구사항
------------------------------------------------------------

1) GET /api/documents
→ 전체 문서 반환

2) GET /api/documents/:id
→ 특정 문서 반환

3) POST /api/documents
→ 새 문서 생성

4) PUT /api/documents/:id
→ 문서 수정

5) DELETE /api/documents/:id
→ 문서 삭제

6) GET /api/search?q=검색어
→ 제목 + 블록 내용 기반 검색 기능 구현

7) POST /api/ai
body:
{
  "content": "문서 내용",
  "action": "summarize | organize | tag"
}

------------------------------------------------------------
6. gemini CLI 실행 규칙
------------------------------------------------------------

명령어 형식:

gemini -y -p "<프롬프트>"

- 반드시 -y 옵션 포함
- 반드시 큰따옴표 사용
- 사용자 입력에 큰따옴표 포함 시 escape 처리
- stdout 결과를 JSON으로 반환

action별 프롬프트:

[summarize]
다음 글을 핵심만 정리하여 요약해줘:

[organize]
다음 글을 가독성 좋게 마크다운 형식으로 구조화해줘:

[tag]
다음 글의 핵심 주제를 기반으로 5개의 태그를 생성해줘. 쉼표로 구분해줘:
